    Nginx 操作命令

    • nginx -s reload 重新加载配置文件
    • nginx -s reopen 重新打开日志文件
    • nginx -t 测试 nginx 配置文件是否正确
    • nginx -t -c /path/to/nginx.conf 测试 nginx 配置文件是否正确
    • nginx -s stop 快速停止 nginx
