shell 操作

  1. 点击 Windows + R 打开在运行界面
  2. 输入powershell 后，打开powershell

在弹出的软件中输入git version,回车查看 git 版本，确认安装成功

确认成功安装后，依次输入以下命令回车执行，使用前面记录的信息替换相关内容，全程一律回车，直到执行完毕。
