  1. 安装 node、yarn、git、vscode

  2. yarn global add hexo-cli

  3. hexo init blog

  4. 配置 git 本地配置

  5. 配置 github 用户 ssh

  6. ssh -T git@github.com

  7. 创建用户名.github.io仓库

  8. git clone git@github.com:用户名/用户名.github.io.git

  9. 将 blog 中所有文件移动至 用户名.github.io文件夹中

  10. 配置 config 中的 deploy 参数

  11. hexo -d

  12. git pull
